# 🎮 Tic-Tac-Toe Game

A simple and responsive Tic-Tac-Toe game built using HTML, CSS, and JavaScript.

## Features

- Two-player mode
- Win detection
- Draw detection
- Restart button
- Responsive UI
- Modern design

## Technologies

- HTML5
- CSS3
- JavaScript

## How to Run

1. Download or clone the repository.
2. Open `index.html` in your browser.
